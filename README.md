# Take-Home Labs: Laboratorios con Arduino Mega (ATmega2560)

## Introducción
Este repositorio contiene una serie de prácticas de laboratorio diseñadas para el autoaprendizaje y la experimentación en casa, enfocadas en la arquitectura del microcontrolador ATmega2560.

## Objetivo General

Desarrollar competencias en programación de sistemas embebidos utilizando la plataforma Arduino Mega, priorizando el control directo del hardware.
